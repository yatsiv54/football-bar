# Football Fan Bar 
![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)

The ultimate hub for football enthusiasts. Browse our menu, book your favorite table, and engage in fan competitions while staying updated with the latest match schedules.

---

## 🚀 Status
- **Google Play Store:** ✅ Verified & Published
- **Current Version:** 1.0.0
- **Build:** Flutter Stable Channel

## 📖 Overview
A sophisticated mobile solution developed for a **Gambling Affiliate Program**. This app serves as a powerful engagement platform, merging real-world venue services with interactive betting and fan-reward ecosystems.

## ✨ Key Features
* **Smart Table Booking:** Interactive seat map for real-time table reservations during major matches.
* **Digital Menu & Ordering:** Complete browse-and-order system for food and beverages.
* **Betting & Predictions:** Seamlessly integrated match betting and fan competition modules.
* **Gamified Challenges:** Participate in fan challenges and earn exclusive rewards and badges.
* **Live Match Schedule:** Real-time updates for upcoming football fixtures and in-bar events.
